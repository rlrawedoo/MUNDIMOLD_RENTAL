Odoo Rental Sale
Odoo rental management system provides the functionality to manage rental products, rental tenure schemes, rental orders, rental contracts, incoming delivery, outgoing delivery etc in an easy and efficient way.

Features:

  • Sales Manager can create Rental Products
  • Every rental products may have several rental tenure scheme with corresponding rent price(like: $20.00 for 15 days, $500.00 for 3 Months etc.)
  • Sales Manager can manage various Rental UOM(like: Days, Months, Years etc.)
  • Security amount facility for each rental product
  • Rental product agreement for each rental product
  • Rental Tenure Type(can be Standard and/or Custom) for each rental product
  • Rental category for each rental product
  • Rental contract for each rental order(or renewal rental order)
  • Manage separate(incoming/outgoing) stock delivery
  • Manage various rental invoices(like: rental contract invoice, security refund invoice)

  • Rental Sale Panel

  • Configuration

Rental Rate Calculation Method: There is two options in rate calculation method.

  1. Simple Calculation:
    In simple calculation method rental amount will be calculated by using smallest rental tenure scheme.
    For example: If product has following rental tenure scheme.
    • Rent for 5.00 days is $500.00, and
    • Rent for 1.00 days is $150.00.
    Now if customer will buy this rental product for 6 days then charges will be calculated based on smallest rental tenure scheme 1.0 day @ $150.00.
    So rent price will 6 * 150.00 / 1.0 = $900.00
  2. Dynamic Calculation:
    In dynamic calculation rental amount will be calculated dynamically. This method first find closest rental tenure scheme then again for remaining tenure and so on.
    For example: If product has following rental tenure scheme.
    • Rent for 5.00 days is $500.00, and
    • Rent for 1.00 days is $150.00.
    Now if customer will buy this rental product for 6 days then charge will be calculated as following:
    For 5.00 days rent will be applied according to 5.00 days @ $500.00 rental scheme, and for remaining tenure
    For 1.00 days rent will be applied according to 1.0 day @ $150.00 rental scheme.
    So final rent price will $650.00

Do not just manage rentals on the Odoo backend but also display rental products on your website with Odoo Website Rental Sale module. Let the customers rent the products on their own directly from the website. The time and resources saved by reducing manual labor in placing the rental order in the backend can be used for other processes.

Key Pointers

  • The module facilitates the creation of products which are available for rent on the Odoo website.
  • The customers can rent the products directly from the website.
  • The Odoo admin can offer both standard and customized tenure for customers.
  • Rental products can be Categorized in the Odoo backend for easier website navigation.
  • Security amount can be mentioned in the product details.
